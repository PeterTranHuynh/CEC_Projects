Begin VI Template Appearance

When first opening the Begin VI and enter it's block diagram, it should appear to look something like the picture to the left. The Begin VI is responsible for intializing each motor, joystick, camera, and other peripheral devices you want to designate to the robot. The template has provided a small sample of code for you. It has already declared a single camera, two motors designated to driving, and a single joystick.

The program written for a camera is already there, and it is as shown here.

You do not need to remove this section of code if you don't intend on using a camara, but it will be useless unless you use one. Our 2012 robot programmers didn't use a webcam for any task, so we will not include webcam programming in this tutorial.
