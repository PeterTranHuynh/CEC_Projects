Get an UPGRADE!

Before starting, you must update all of your resources. Make sure you have all software libraries downloaded and set-up. You can find update clients here. Just look for the Software Downloads link in the FRC section of the website. After installing the updates, you can begin setting up

Getting Started

Once you've installed all the required updates and libraries, you can start programming in LabVIEW.

Open up labview, and you should get something like this.

Choose the template FRC cRIO Robot project, this will provide you with all the required VI program files you will load into the Robot's cRIO unit.

 

After selecting the template for you project, you will be prompted with this screen.

In this window, you can set your project name, location, and, most importantly, the cRIO IP address. The address for your cRIO commonly consists of a 10, your team numbers, and single digit. For example, our 2012 Robot, Jarvis, had 10.22.59.2 as the cRIO IP address.

After setting up your project, you will be moved to the main LabVIEW window screen:

This window will display all VI files, library folders, and build dependencies. Here, you will see which files are responsible for the robot's processing and functions. Each VI file handles a different task, for example Teleop handles user input and movements and Autonomous handles independent robot movement.
